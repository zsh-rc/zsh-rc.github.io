### 1.哀 
#### （1）哀鸣 
风急天高猿啸哀（《登高》） <br/>
未至二三里，摧藏马悲哀（《孔雀东南飞并序》） <br/>
#### （2）哀叹 
哀吾生之须臾，羡长江之无穷（《赤壁赋》） <br/>
#### （3）声音凄清尖利地 
杜鹃啼血猿哀鸣（《琵琶行并序》） <br/>
#### （4）同情 
长太息以掩涕兮，哀民生之多艰（《离骚（节选）》） <br/>
#### （5）为……悲哀 
秦人不暇自哀，而后人哀之（《阿房宫赋》） <br/>
亦使后人而复哀后人也（《阿房宫赋》） <br/>
### 2.爱 
#### （1）爱护 
爱其子，择师而教之；于其身也，则耻师焉，惑矣（《师说》） <br/>
苟有能反是者，则又爱之太恩，忧之太勤（《种树郭橐驼传》） <br/>
虽曰爱之，其实害之（《种树郭橐驼传》） <br/>
察此何自起？皆起不相爱（《兼爱》） <br/>
盗爱其室，不爱异室，故窃异室以利其室（《兼爱》） <br/>
父自爱也，不爱子，故亏子而自利（《兼爱》） <br/>
故圣人以治天下为事者，恶得不禁恶而劝爱（《兼爱》） <br/>
若使天下兼相爱，爱人若爱其身，犹有不孝者乎（《兼爱》） <br/>
此四君者，皆明智而忠信，宽厚而爱人（《过秦论》） <br/>
上于盆而养之，蟹白栗黄，备极护爱，留待限期，以塞官责（《促织》） <br/>
会有四方之事，诸侯以惠爱为德（《归去来兮辞并序》） <br/>
#### （2）爱恋 
吾爱汝至，所以为汝谋者惟恐未尽（《与妻书》） <br/>
吾充吾爱汝之心，助天下人爱其所爱，所以敢先汝而死，不顾汝也（《与妻书》） <br/>
吾至爱汝，即此爱汝一念，使吾勇于就死也（《与妻书》） <br/>
#### （3）爱惜 
使六国各爱其人，则足以拒秦（《过秦论》） <br/>
向使三国各爱其地，齐人勿附于秦（《六国论》） <br/>
#### （4）吝啬，舍不得 
百姓皆以王为爱也，臣固知王之不忍也（《齐桓晋文之事》） <br/>
王无异于百姓之以王为爱也（《齐桓晋文之事》） <br/>
我非爱其财而易之以羊也，宜乎百姓之谓我爱也（《齐桓晋文之事》） <br/>
齐国虽褊小，吾何爱一牛（《齐桓晋文之事》） <br/>
诸侯恐惧，会盟而谋弱秦，不爱珍器重宝肥饶之地，以致天下之士，合从缔交，相与为一（《过秦论》） <br/>
#### （5）喜爱，喜欢 
三五之夜，明月半墙，桂影斑驳，风移影动，珊珊可爱（《项脊轩志》） <br/>
少无适俗韵，性本爱丘山（《归园田居（其一）》） <br/>
秦爱纷奢，人亦念其家（《阿房宫赋》） <br/>
武能网纺缴，檠弓弩，於靬王爱之，给其衣食（《苏武传》） <br/>
#### （6）同“薆”，隐藏 
爱而不见，搔首踟蹰（《静女》） <br/>
### 3.安 
#### （1）安处 
何方圜之能周兮，夫孰异道而相安（《离骚（节选）》） <br/>
#### （2）安定，使……安定 
思国之安者，必积其德义（《谏太宗十思疏》） <br/>
其安易持，其未兆易谋（《老子》四章） <br/>
吾小人辍飧饔以劳吏者，且不得暇，又何以蕃吾生而安吾性耶（《种树郭橐驼传》） <br/>
君人者，诚能见可欲则思知足以自戒，将有作则思知止以安人（《谏太宗十思疏》） <br/>
#### （3）安全 
且陛下春秋高，法令亡常，大臣亡罪夷灭者数十家，安危不可知，子卿尚复谁为乎（《苏武传》） <br/>
不念居安思危，戒奢以俭（《谏太宗十思疏》） <br/>
#### （4）安适，安稳 
时时为安慰，久久莫相忘（《孔雀东南飞并序》） <br/>
倚南窗以寄傲，审容膝之易安（《归去来兮辞并序》） <br/>
予出官二年，恬然自安，感斯人言，是夕始觉有迁谪意（《琵琶行并序》） <br/>
君子食无求饱，居无求安，敏于事而慎于言（《论语·学而》） <br/>
今日割五城，明日割十城，然后得一夕安寝（《六国论》） <br/>
#### （5）哪里 
固一世之雄也，而今安在哉（《赤壁赋》） <br/>
沛公安在（《鸿门宴》）<br/>
终不得归汉，空自苦亡人之地，信义安所见乎（《苏武传》） <br/>
#### （6）怎么 
安见方六七十，如五六十而非邦也者（《子路、曾皙、冉有、公西华侍坐》） <br/>
安能摧眉折腰事权贵，使我不得开心颜（《梦游天姥吟留别》） <br/>
臣死且不避，卮酒安足辞（《鸿门宴》） <br/>
君安与项伯有故（《鸿门宴》） <br/>
人生亦有命，安能行叹复坐愁（鲍照（《拟行路难（其四）》） <br/>
又安能以皓皓之白，而蒙世之温蠖乎（《屈原列传》） <br/>
#### （7）性情安和 
知止而后有定；定而后能静（《大学之道》） <br/>

### 8.拔 
#### (1) 选取，提拔 
过蒙拔擢，宠命优渥，岂敢盘桓，有所希冀。（《陈情表》） <br/>
#### (2) 攻取 
惠王用张仪之计，拔三川之地（《谏逐客书》） <br/>
#### (3) 超出 
天姥连天向天横，势拔五岳掩赤城。（《梦游天姥吟留别》） <br/>

樊哙覆其盾于地，加彘肩上，拔剑切而啖之。（《鸿门宴》） <br/>
