### 289.再 
#### （1）两次 
后秦击赵者再，李牧连却之（《六国论》） <br/>
正旦再跪科（《窦娥冤》）节选） <br/>
庭中始为篱，已为墙，凡再变矣（《项脊轩志》） <br/>
谨使臣良奉白璧一双，再拜献大王足下（《鸿门宴》）再拜：拜两拜，古代一种较重的礼节 <br/>
问者爇香于鼎，再拜（《促织》）再拜：拜两拜，古代一种较重的礼节 <br/>
府吏默无声，再拜还入户（《孔雀东南飞并序》）再拜：拜两拜，古代一种较重的礼节 <br/>
#### （2）第二次 
一鼓作气，再而衰，三而竭（《曹刿论战》） <br/>
#### （3）接着，用在两个动词之间，表示动作的先后关系 
你只去听一听，再理会（《林教头风雪山神庙》） <br/>
再说林冲踏着那瑞雪（《林教头风雪山神庙》） <br/>
入得庙门，再把门掩上（《林教头风雪山神庙》） <br/>
再穿了白布衫，系了褡膊（《林教头风雪山神庙》） <br/>
#### （4）事情或行为重复、继续 
再也不要啼啼哭哭（《窦娥冤》）节选） <br/>
再看一看，拾得他一两块骨头回京（《林教头风雪山神庙》） <br/>
子为父死，亡所恨，愿无复再言（《苏武传》） <br/>
#### （5）重，重新 
约计吃过十数杯，再讨了按酒铺放桌上（《林教头风雪山神庙》） <br/>
#### （6）更，更加 ，表示程度 
却被石头靠住了，再也推不开（《林教头风雪山神庙》） <br/>
### 290.造

取之无禁，用之不竭，是造物者之无尽藏也（《赤壁赋》） <br/>
造物者：自然界。原指“天”，就是现在所说的“自然”。 <br/>
#### （2）到 
径造庐访成（《促织》） <br/>
#### （3）做，行 
为善的受贫穷更命短，造恶的享富贵又寿延。（《窦娥冤》） <br/>
### 291.责 
#### （1）责备，指责 
如君实责我以在位久，未能助上大有为（王安石《答司马谏议书》） <br/>
诏书切峻，责臣逋慢。郡县逼迫，催臣上道（李密《陈情表》） <br/>
#### （2）索取 
里胥猾黠，假此科敛丁口，每责一头，辄倾数家之产。（蒲松龄《促织》） <br/>
#### （3）派发责任、差事 
令以责之里正（蒲松龄《促织》）（此句翻译：县令把进贡的差事派给里长） <br/>
#### （4）责令

### 292.贼 
#### （1）犯上作乱的人，外来侵犯的人。 
第以今日事势观之，天灾可以死，盗贼可以死，瓜分之日可以死（《与妻书》） <br/>
泼贼那里去！（《林教头风雪山神庙》） <br/>
贼爱其身，不爱人，故贼人以利其身。（《兼爱》） <br/>
那泼贱贼敢来这里害我!（《林教头风雪山神庙》）泼贱贼：歹毒无赖的奸贼 <br/>
汉家烟尘在东北，汉将辞家破残贼。（《燕歌行并序》） 残贼：指残忍暴虐的敌寇 <br/>
仓皇东出，未及见贼而士卒离散（《五代史伶官传序》） 对敌人蔑称 <br/>
#### （2）盗贼：偷窃和劫夺财物的人 
虽至天下之为盗贼者，亦然。（《兼爱》） <br/>
故盗贼有亡。（《兼爱》） <br/>
盗贼无有，君臣父子皆能孝慈，若此则天下治。（《兼爱》） <br/>
#### （3）伤害 
有是四端而自谓不能者，自贼者也；谓其君不能者，贼其君者也。（《孟子·人皆有不忍人之心》） <br/>
#### （4）作乱叛国危害他人

### 297.直

木直中绳，輮以为轮，其曲中规。（《劝学》） <br/>
故木受绳则直，金就砺则利（《劝学》） <br/>
直栏横槛，多于九土之城郭（《阿房宫赋》） <br/>
#### （2）一直 
骊山北构而西折，直走咸阳（《阿房宫赋》） <br/>

#### （4）行为正直

### 300.治
