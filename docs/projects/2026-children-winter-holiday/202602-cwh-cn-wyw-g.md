### 100.盖 
#### （1）句首语气词。表示要发议论 
盖将自其变者而观之（《赤壁赋》） <br/>
盖叹郦元之简，而笑李渤之陋也（《石钟山记》） <br/>
#### （2）同“盍”，何、怎么 
技盖至此乎（《庖丁解牛》） <br/>
#### （3）副词，大概，一般 
能克终者盖寡（《谏太宗十思疏》） <br/>
盖不能以寸管形容之（《与妻书》） <br/>
#### （4）覆盖，遮盖 
将火炭盖了（《林教头风雪山神庙》） <br/>
#### （5）连词。大概因为。说明原因，并带有测度的意味 
盖自怨生也（《屈原列传》） <br/>
吾之意盖谓以汝之弱（《与妻书》） <br/>
盖儒者所争，尤在于名实（《答司马谏议书》） <br/>
#### （6）伞盖 
今已亭亭如盖矣（《项脊轩志》） <br/>

梧桐更兼细雨（《声声慢（寻寻觅觅）》） <br/>
为善的受贫穷更命短（《窦娥冤》） <br/>
#### （3）改变、更换

更无休止（《促织》） <br/>
更无一点风色（《念奴娇》） <br/>

夜夜达五更（《孔雀东南飞并序》） <br/>
#### （6）交替，轮换。（更相为相互） 
母、孙二人，更相为命（《陈情表》） <br/>
### 103.苟

苟以天下之大（《六国论》） <br/>

### 106.故 
#### （1）缘故，原因

暮去朝来颜色故（《琵琶行并序》）（旧，老） <br/>
故垒西边（《念奴娇·赤壁怀古》） <br/>
故国不堪回首月明中（《虞美人·春花秋月何时了》） <br/>
君安与项伯有故（《鸿门宴》）（旧，交情） <br/>
亲故多劝余为长吏（《归去来兮辞并序》）（故旧，老友） <br/>
#### （3）连词。所以，因此

故能就其深（《谏逐客书》） <br/>
