争宠而心害其能（《屈原列传》） <br/>
才能不及中人（《过秦论》） <br/>
#### （2）有才能的人 
简能而任之（《谏太宗十思疏》） <br/>
#### （3）会，善于 
非能水也，而绝江河（《劝学》） <br/>
#### （4）胜任，能做到 
非曰能之，愿学焉（《侍坐章》） <br/>
不为也，非不能也（《齐桓晋文之事》） <br/>
无恒产而有恒心者，惟士为能（《齐桓晋文之事》） <br/>
#### （5）能够 
安能摧眉折腰事权贵（《梦游天姥吟留别》） <br/>
尚能饭否（《永遇乐·京口北固亭怀古》） <br/>
人非生而知之者，孰能无惑（《师说》） <br/>
然不自意能先入关破秦（《鸿门宴》） <br/>
杀人如不能举（《鸿门宴》） <br/>
不能竟书而欲搁笔（《与妻书》） <br/>
并虫亦不能行捉矣（《促织》） <br/>
#### （6）及，到 
骐骥一跃，不能十步（《劝学》） <br/>
则天地曾不能以一瞬（《赤壁赋》） <br/>
#### （7）通“态” 
又重之以修能（《离骚（节选）》） <br/>
