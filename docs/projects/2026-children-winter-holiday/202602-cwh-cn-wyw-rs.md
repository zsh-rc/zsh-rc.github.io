### 198.任 
#### （1）担子、任务 
士不可以不弘毅，任重而道远（《论语·泰伯》） <br/>
#### （2）放纵，不加约束： 
寓形宇内复几时，曷不委心任去留（《归去来兮辞并序》） <br/>
恐不任我意（《孔雀东南飞并序》） 听凭，任凭 <br/>
#### （3）委任，任用 
少以父任，兄弟并为郎（《苏武传》） <br/>
简能而任之，择善而从之（《谏太宗十思疏》） <br/>
#### （4）相信，信任 
王甚任之（《屈原列传》） <br/>
#### （4）堪，承当，禁受 
无由会晤，不任区区向往之至（《答司马谏议书》） <br/>
### 199.忍 
#### （1）忍耐，忍受 
屈心而抑志兮，忍尤而攘诟（《离骚（节选）》） <br/>
#### （2）忍心，狠心，残忍 
人皆有不忍人之心（《人皆有不忍人之心》） <br/>
吾不忍其觳觫（《齐桓晋文之事》） <br/>
君王为人不忍（《鸿门宴》） <br/>
卒不忍独善其身（《与妻书》） <br/>
#### （3）把感情按住不让表现，克制情感 
故遂忍悲为汝言之（《与妻书》） <br/>
#### （4）怎，岂 
忍顾鹊桥归路（《鹊桥仙》） <br/>
### 200.如 
#### （1）遵从，按照，依照 
如惠语以让单于（《苏武传》） <br/>
#### （2）好像， 如同 
譬如朝露，去日苦多。（《短歌行》） <br/>
闻道有先后，术业有专攻，如是而已。（《师说》） <br/>
浩浩乎如冯虚御风（《赤壁赋》） <br/>
如怨如慕，如泣如诉（《赤壁赋》） <br/>
謋然已解，如土委地（《庖丁解牛》） <br/>
杀人如不能举，刑人如恐不胜（《鸿门宴》） <br/>
闻妻言，如被冰雪（《促织》）  <br/>
#### （3）比得上，及 
是故弟子不必不如师（《师说》） <br/>
他植者虽窥伺效慕，莫能如也（《种树郭橐驼传》） <br/>
#### （4）或——表示选择关系 
方六七十，如五六十（《侍坐》） <br/>
宗庙之事，如会同，端章甫（《侍坐》） <br/>
#### （5）假如，如果 
如或知尔，则何以哉（《侍坐》） <br/>
#### （6）去，往 
纵一苇之所如（《赤壁赋》） <br/>
沛公起如厕（《鸿门宴》） <br/>
臣请往如楚（《屈原列传》） <br/>
#### （7）【如其】连词，至于。表示提出另一话题，用在下句或下段开头 
如其礼乐，以俟君子（《侍坐》） <br/>
#### （8）【如何】怎么样 
尔何如？（《侍坐》） <br/>
### 201.入 
#### （1）进入 
阳谷皆入汶（《登泰山记》） <br/>
备他盗之出入与非常也（《鸿门宴》） <br/>
以深入击秦（《屈原列传》） <br/>
入于太庙，还矢先王（《五代史伶官传序》） <br/>
鸡鸣入机织，夜夜不得息（《孔雀东南飞并序》） <br/>
以无厚入有间（《庖丁解牛》） <br/>
#### （2）掉入、坠入 
今人乍见孺子将入于井（《人皆有不忍人之心》） <br/>
景翳翳以将入（《归去来兮辞并序》） <br/>
#### （3）参加，加入 
又嘱学使俾入邑庠（《促织》） <br/>
#### （4）对内，在国内 
入则与王图议国事（《屈原列传》） <br/>
#### （5）接纳，采纳 
进不入以离尤兮（《离骚（节选）》） <br/>
#### （6）入朝，指属国、外国使臣或地方官员谒见天子 
强国请服，弱国入朝（《过秦论》） <br/>
项伯即入见沛公（《鸿门宴》） <br/>
#### （7）收入，进项 
殚其地之出，竭其庐之入（《捕蛇者说》） <br/>
### 202.若 
#### （1）如同，像 
而半山居雾若带然（《登泰山记》） <br/>
若无罪而就死地（《齐桓晋文之事》） <br/>
覆之以掌，虚若无物（《促织》） <br/>
雍州之地，崤函之固，自若也（《过秦论》） <br/>
汩余若将不及兮（《离骚（节选）》） <br/>
#### （2）同，相当 
彼与彼年相若也（《师说》） <br/>
#### （3）及，比得上 
其实仇之故不我若也（《种树郭橐驼传》） <br/>
#### （4）如此，这样 
君既若见录（《孔雀东南飞并序》） <br/>
#### （5）你 
以若所为，求若所欲（《齐桓晋文之事》） <br/>
若入前为寿，寿毕，请以剑舞（《鸿门宴》） <br/>
若知我不降明（《苏武传》） <br/>
【若属】你们 <br/>
若属皆且为所虏（《鸿门宴》） <br/>
#### （6）假如，如果 
两情若是久长时（《鹊桥仙》） <br/>
王若隐其无罪而就死地（《齐桓晋文之事》） <br/>
若不阙秦，将焉取之（《烛之武退秦师》） <br/>
若不过焉则不及（《种树郭橐驼传》） <br/>
#### （7）用在形容词或副词后面，表示事物的状态。 
桑之未落，其叶沃若（《氓》） <br/>
#### （8）至于。用在句首以引起下文 
若夫霪雨霏霏（《岳阳楼记》） <br/>
### 203.塞

### 204.色 
#### （1）脸色 
成仓猝莫知所救，顿足失色（《促织》） <br/>
#### （2）姿色，容颜 
暮去朝来颜色故（《琵琶行并序》） <br/>
颜色憔悴，形容枯槁（《屈原列传》） <br/>
#### （3）色彩，颜色 
血色罗裙翻酒污（《琵琶行并序》） <br/>
极天云一线异色（《登泰山记》） <br/>
江天一色无纤尘（《春江花月夜》）  <br/>
#### （4）风景，景观 
目遇之而成色（《赤壁赋》） <br/>
抑为采色不足视于目与（《齐桓晋文之事》） <br/>
#### （5）色欲，男女爱情 
然则是所重者在乎色、乐、珠玉（《谏逐客书》） <br/>
《国风》）好色而不淫（《屈原列传》） <br/>
#### （6）物质（多指金银）的成分 
天子非常赐颜色（《燕歌行》） <br/>
#### （7）天色 
怎么这一会儿天色阴了也（《窦娥冤》） <br/>
暮色渐起，戍角悲吟（《扬州慢（并序）》） <br/>
### 205.善 
#### （1）好，美好 
然后驱而之善（《齐桓晋文之事》） <br/>
有善始者实繁（《谏太宗十思疏》） <br/>
士大夫多以不恤国事、同俗自媚于众为善（《答司马谏议书》） <br/>
非兵不利，战不善（《六国论》） <br/>
亦余心之所善兮（《离骚（节选）》） <br/>
#### （2）高明，工巧 
曲罢曾教善才服（《琵琶行并序》） <br/>
#### （3）好好地 
不如因善遇之（《鸿门宴》） <br/>
汝其善抚之，使之肖我（《与妻书》） <br/>
#### （4）好人，好事，好处 
积善成德，而神明自得（《劝学》） <br/>
为善的受贫穷更命短（《窦娥冤》） <br/>
择善而从之（《谏太宗十思疏》） <br/>
#### （5）善于，擅长。有做好或处理好某事的才能或技巧 
善假于物也（《劝学》） <br/>
善推其所为而已矣（《齐桓晋文之事》） <br/>
自言身化促织，轻捷善斗（《促织》） <br/>
#### （6）使……好 
卒不忍独善其身（《与妻书》） <br/>
#### （7）通“缮”。修治 
善刀而藏之（《庖丁解牛》） <br/>
#### （8）羡慕 
善万物之得时，感吾生之行休（《归去来兮辞并序》） <br/>
#### （9）表示应诺。对，好 
善哉！技盖至此乎？（《庖丁解牛》） <br/>
#### （10）与……友好，交好，亲善 
素善留侯张良（《鸿门宴》） <br/>
### 206.稍

### 207.少 
#### （1）与“多”相对 
一时多少豪杰（《念奴娇·赤壁怀古》） <br/>
余始循以入，道少半（《登泰山记》） <br/>
主人何为言少钱（《将进酒》） <br/>
及少日，眷然有“归欤”之情。（《归去来兮辞并序》） <br/>
#### （2）无 
你只看窦娥少爷无娘面（《窦娥冤》） <br/>
#### （3）年幼，年轻 
少无适俗韵（《归园田居（其一）》） <br/>
夜深忽梦少年事（《琵琶行并序》） <br/>
是故无贵无贱，无长无少（《师说》） <br/>
沛公曰：“孰与君少长?”（《鸿门宴》） <br/>
少以父任，兄弟并为郎（《苏武传》） <br/>
#### （4）年轻人 
五陵年少争缠头（《琵琶行并序》） <br/>
群贤毕至，少长咸集（《兰亭集序》） <br/>
#### （5）稍微，短暂 
且请少坐（《林教头风雪山神庙》） <br/>
解鞍少驻初程（《扬州慢》） <br/>
#### （6）一会儿，时间少 
少焉，月出于东山之上（《赤壁赋》） <br/>
有书在此，少刻便知（《林教头风雪山神庙》） <br/>
少间，帘内掷一纸出（《促织》） <br/>
#### （7）【少牢】旧时祭礼的牺牲﹐只用羊﹑豕二牲叫少牢 
则遣从事以一少牢告庙（《五代史伶官传序》） <br/>
### 208.舍

### 211.甚

### 213.胜 
#### （1）禁得起 
沛公不胜杯杓（《鸿门宴》） <br/>
更恐不胜悲（《与妻书》） <br/>
青枫浦上不胜愁（《春江花月夜》） <br/>
臣不胜犬马怖惧之情（《陈情表》） <br/>
#### （2）胜过，超过  
此时无声胜有声（《琵琶行并序》） <br/>
便胜却人间无数（《鹊桥仙》） <br/>
质胜文则野；文胜质则史（〈论语〉十二章》） <br/>

#### （4）克制，制服

#### （5）胜利 
故不战而强弱胜负已判矣（《六国论》） <br/>
#### （6）美好，美妙 
予观夫巴陵胜状（《岳阳楼记》） <br/>
#### （7）美景，名胜古迹
