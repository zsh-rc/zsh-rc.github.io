### 83.恶

以小易大，彼恶知之？（《齐桓晋文之事》） <br/>
视父兄与君若其身，恶施不孝？（《兼爱》） <br/>

无羞恶之心，非人也。（《人皆有不忍人之心》） <br/>

非恶其声而然也。（《人皆有不忍人之心》） <br/>
物或恶之，故有道者不处也。（《〈老子〉四章》） <br/>
故天下兼相爱则治，交相恶则乱。（《兼爱》） <br/>

### 85.伐 
#### （1）砍伐 
斯亦伐根以求木茂，塞源而欲流长者也。（《谏太宗十思疏》） <br/>

每一令出，平伐其功，曰以为‘非我莫能为’也。（《屈原列传》） <br/>

屈原既绌，其后秦欲伐齐，齐与楚从亲。（《屈原列传》） <br/>
时汉连伐胡，数通使相窥观。（《苏武传》） <br/>
#### （4）敲打 
𪭢金伐鼓下榆关，旌旆逶迤碣石间。（《燕歌行》） <br/>
